> 社会，就如同人的身体，拥有相互关联的部分、需要和功能。 ——埃米尔·涂尔干

---

现代社会是高度合作的社会。无论是哪个行业，都需要遵循一定的标准规范。使用他人提供的工具、遵循他人制定的流程，来完成个体的工作。

在软件开发领域，通常管这个叫技术栈。但对我来说，这更像是由一次次实践累积起来的“实践指南”，于是我打算记录下自己使用过的这些由他人提供的工具和流程。

这些实践指南是高度**片面**、**个人化**且**实用主义**的，包含了我个人的粗略理解、大量的使用案例、各式各样的问题记录，当作手册来随时查阅、增改。
